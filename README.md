## Installation

```bash
# npm
$ npm install

# yarn
$ yarn
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Build

```bash
# dev
yarn build:dev

#prod
yarn build:prod

```
